## 새로 배운 내용
dbp를 하려면 4가지를 선택해야함을 배웠다.
1. 호스트 언어를 무엇으로 할 것인가? PHP, JAVA
2. DBMS를 무엇으로 할 것인가? MariaDB, MySQL, Oracle
3. 어떤 운영체제를 사용할 것인지
4. 컴퓨터 기종을 뭘 선택할건지

- 이번 시간에는 Java와 Oracle을 연동하여 환경을 구축하는 실습을 진행했다.
- JDBC는 자바와 데이터베이스를 연동할 수 있는 API이다.
- JDBC 프로그래밍 과정

        1) JDBC 드라이버 로드
        2) DB 연결
        3) DB에 데이터 읽기 쓰기(SQL 구문)
        4) DB 연결 종료
